#  Математические формулы и как использовать функции
## Площадь
### Круг: S = πR²
- Используйте когда вам нужно посчитать площадь круга и вам известен радиус. Например:
`print(area(5))` - посчитает площадь для круга с радиусом = 5

### Прямоугольник: S = ab
- Используйте, когда вам нужно вычислить площадь прямоугольника, и вы знаете две разные стороны. Например:
`print(area(4, 7))` - вычислит площадь прямоугольника со стороной a = 4 и стороной b = 7

### Квадрат: S = a²
- Используйте, когда вам нужно вычислить площадь квадрата и вы знаете его сторону. Например:
`print(area(6))` - вычислит площадь для квадрата со стороной = 6


## Периметр
### Круг: P = 2πR
- Используйте, когда вам нужно рассчитать периметр круга и вы знаете радиус. Например:
`print(perimeter(5))` - вычислит периметр для окружности с радиусом = 5

### Прямоугольник: P = 2a + 2b
- Используйте, когда вам нужно рассчитать периметр прямоугольника, и вы знаете две разные стороны. Например:
`print(perimeter(2, 7))` - вычислит периметр прямоугольника со сторонами a = 2 и b = 7

### Квадрат: P = 4a
- Используйте, когда вам нужно рассчитать периметр квадрата и вы знаете его сторону. Например:
`print(perimeter(5))` - вычислит периметр для квадрата со стороной = 5
